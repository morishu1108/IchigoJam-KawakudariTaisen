# IchigoJam-KawakudariTaisen
Source code for customized "kawakudari" game.You can play with 2 players using a keyboard.
川くだりゲームの改造版。1つのキーボードを使って2人のプレイヤーが対戦可能となっている。

# プレイ方法
1P:"Z"で左移動、"C"で右移動  
2P:"←"で左移動、"→"で右移動

# 改造例
* キャラクターを独自のキャラクターにしてもよいかも  
* ポイント表示をいれてみてもよいかも
* 3人プレイにしてみてもよかも
